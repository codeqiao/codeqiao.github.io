知识点1 打印数组

打印int等基本数据类型变量会直接打印结果，打印引用数据类型一般情况下会打印地址值，但是当打印char型的数组时会直接打印所有的字符。

知识点2 JavaBean

满足如下标准的java类被称为JavaBean

  • 类需要用public修饰

  • 必须要有一个无参数的用public修饰的构造器

  • 有属性，且需要有相应的get，set方法
