打印int等基本数据类型变量会直接打印结果，打印引用数据类型一般情况下会打印地址值，但是当打印char型的数组时会直接打印所有的字符。

知识点2 JavaBean

满足如下标准的java类被称为JavaBean

  • 类需要用public修饰

  • 必须要有一个无参数的用public修饰的构造器

  • 有属性，且需要有相应的get，set方法

知识点6 接口

  • 接口中定义的静态方法，只能通过接口来调用

  • 通过实现类的对象可以调用接口中的默认方法

  • 如果子类继承的父类和实现的接口中声明了同名同参数的方，那么在子类没有重写此方法的情况下，默认调用的是父类中同名同参数的方法。

  • 子类中调用接口中被重写的默认方法要用接口名.super.方法名。

知识点9 异常

  • error

  • Exception ：编译时异常，运行时异常

两种方法处理异常

  • 在try中定义的变量出了try之后就不能在使用了

  • finally可以省略

异常对象常用的两个方法

体会：

  • 想数据库连接，输入输出流，网络套接字，JVM 的垃圾回收机制没有用，需要手动释放，这是就会用到finally

  • 开发中，运行时异常一般不会try catch，编译时的异常才会用

  • 子类重写的方法抛出的异常不大于父类被重写抛出的异常类型

  • 开发中如何选择这两种处理异常的方法

    1.子类重写的方法中有异常，父类没有异常，只能用try catch

    2.执行的方法中又先后执行另外几个方法，这几个方法时递进执行的，那么这几个方法用throws，调用的方法用try-catch来处理。
